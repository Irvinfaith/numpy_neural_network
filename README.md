# 1. 概览

本项目主要用于神经网络的学习，通过基于numpy的实现，了解神经网络底层前向传播、反向传播的原理。

该项目目前已实现的功能：

- 自定义多层的全连接层，并可定义多种激活函数
  - sigmoid
  - tanh
  - relu
- 定义dropout
- 支持多种优化器
  - SGD
  - BSGD
  - SGD with Momentum
  - AdaGrad
  - AdaDelta
  - RMSProp
  - Adam
  - AdaMax
